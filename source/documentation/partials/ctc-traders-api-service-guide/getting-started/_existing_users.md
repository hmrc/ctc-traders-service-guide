Traders upgrading from NCTS5 (v2.1) to NCTS6 (v3) and familiar with CTC Traders API v2.1 must take the following actions:


- Review this section prior to consulting other NCTS6 documentation.
- Determine if Trader data applies to any traders served, as affected traders will need to act accordingly.
- Review the P6 Technical Interface Specification (TIS) for updated content.
- Review the guidelines for making API requests.
- Evaluate the test scenarios for Phase 6.
